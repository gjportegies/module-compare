# Swissup Compare

    `Magento2 extension`

### Features

 *  Changing default magento **Products compare** page look
 *  Adding option to **Disable** compare functionality in **Magento**

### Admin location

    `Swissup > Compare`

### Options

 *  Enable **Alternative "Compare page"** design
    _(Default = NO)_
 *  Hide default Magento **"Compare"** functionality
    _(Default = NO)_
